# 3人子育てドタバタ母さんブログ 収益化ロードマップ

作成日: 2026-07-07
現状: 公開約65記事 / 下書きストック46本 / 月木自動公開 / インスタ週末投稿 / 楽天アフィリエイト導入済み

---

## 全体像

エッセイ中心のブログは「読者に愛される」タイプで、検索流入だけで大きく稼ぐタイプではない。
そのため収益の柱を3つに分けて育てる:

1. **広告収入(AdSense)** — PVに比例。まずは土台
2. **アフィリエイト** — 「役立ち系記事」で稼ぐ。単価が高く本命
3. **インスタ導線** — エッセイのファンを増やしブログへ送客

---

## フェーズ1: 計測の土台づくり(今月中)

収益化の前に「何人読んでいるか」が見えないと打ち手が決められない。

- [ ] Google Analytics 4 導入(hugo.tomlに設定追加するだけ。PaperMod対応済み)
- [ ] Google Search Console 登録 + sitemap.xml送信
- [ ] 現状のPV/月を記録(ベースライン)

**ゴール: 月間PVと検索キーワードが見える状態**

## フェーズ2: AdSense合格(1〜2ヶ月)

記事数65本は審査に十分。不足しているのは定型ページ。

- [ ] プライバシーポリシーページ作成(アフィリエイト表記・Cookie・GA利用を明記)
- [ ] お問い合わせページ作成(Googleフォーム埋め込みで可)
- [ ] 運営者情報(About)ページ作成
- [ ] AdSense申請 → 合格後、広告を記事内に設置

**ゴール: 広告収入の仕組みが動く(最初は月数十〜数百円で正常)**

## フェーズ3: 「稼ぐ記事」を月1〜2本追加(3〜6ヶ月)

エッセイは月木のまま続け、**別枠で検索向けの役立ち記事**を書く。
これが既存の改善案「月1回の役立ち情報まとめ」の収益化版。

ネタ候補(実体験があるので書きやすい):
- 3人子育てで本当に使った育児グッズレビュー(楽天リンク)
- 「小学生 朝の支度 時短」「兄弟げんか 対処」などの検索キーワード向けハウツー
- 季節もの(入学準備・夏休みの過ごし方・クリスマスプレゼント)は2〜3ヶ月前に公開

あわせて:
- [ ] もしもアフィリエイト登録(Amazon+楽天を1リンクにできる「かんたんリンク」が使える)
- [ ] エッセイ記事から関連する役立ち記事へ内部リンクを張る

**ゴール: 検索流入で読まれる記事が5〜10本、アフィリエイト初発生**

## フェーズ4: 伸びたものを増やす(6〜12ヶ月)

- Search Consoleで表示回数が多い記事をリライト(タイトル改善・情報追加)
- インスタのフォロワー増加施策(リール化など)、楽天ROOM併用も検討
- 収益が出た記事の「型」で横展開

**ゴール: 月5,000円 → 月10,000円**

## 収益目安(現実的なライン)

| 時期 | 目標 |
|---|---|
| 〜3ヶ月 | AdSense合格・計測開始(収益ほぼ0でOK) |
| 3〜6ヶ月 | 月500〜1,000円 |
| 6〜12ヶ月 | 月3,000〜10,000円 |
| 1年〜 | 役立ち記事の蓄積次第で月1〜3万円 |

※副業ブログの収益は「1年目はほぼ育成期間」が普通。今の自動化された更新体制を止めないことが最大の武器。
